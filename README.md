# Students Grievance Management System

## Description
The **Students Grievance Management System** is a web-based application that allows students to submit complaints and track their resolution. Admins and employees can manage complaints efficiently. This system simplifies grievance handling in educational institutions.

---

## Features
- **Students:**
  - Register and login
  - Submit complaints
  - Track complaint status
- **Admin/Employees:**
  - View and update complaint status
  - Add employees
  - Archive resolved complaints
- **Security:**
  - Role-based access
  - Authentication for authorized users only

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Tools:** Git, GitHub, XAMPP

---

## Installation

1. Clone the repository:

git clone https://github.com/rutujabobade2133/Students-Grievance-Management-System.git

2. Navigate to the project folder:
cd Students-Grievance-Management-System


3. Run the server:
   - If using XAMPP: place the project folder in `htdocs` and start **Apache** & **MySQL**.
   - Open in browser: `http://localhost/Students-Grievance-Management-System/`

4. Setup Database:
   - Create a MySQL database (e.g., `grievance_system`).
   - Import the provided `database.sql` file if available.
   - Update the database credentials in `config.inc.php` (or equivalent).

---

## Usage
1. Open the project in a browser.
2. Students can register and submit complaints.
3. Admins and employees can log in to manage complaints.
4. Track complaint status and view resolved complaints.
